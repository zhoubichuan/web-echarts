## 3.git submodule

项目仓库

```sh
.
└── main # 项目仓库文件夹
```

共享代码仓库

```sh
.
└── sub  # 共享代码仓库文件夹
```

项目仓库下实现共享

```sh
git submodule add [共享代码仓库地址]
```

完成共享

```sh
.
├── main # 项目仓库文件夹
└── share   #共享代码仓库
      └── sub
```
