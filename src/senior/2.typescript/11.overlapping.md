# 十一.交叉类型

::: tip 前言
交叉类型是将多个类型合并为一个类型
:::

```ts
interface Person1 {
  handsome: string;
}
interface Person2 {
  high: string;
}
type P1P2 = Person1 & Person2;
let p: P1P2 = { handsome: "帅", high: "高" };
```

- 交叉类型

```ts
function mixin<T, K>(a: T, b: K): T & K {
  return { ...a, ...b };
}
const x = mixin({ name: "abc" }, { age: 11 });
```

```ts
interface IPerson1 {
  name: string;
  age: number;
}
interface IPerson2 {
  name: string;
  age: number;
}
type person = IPerson1 & IPerson2;
let name!: never;
let person: person = { name, age: 11 }; //两个属性之间string & number的值为never
```
