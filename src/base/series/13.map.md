# 地理坐标/地图

::: tip 遇到的难点

- 问题 1:3D 地图中平移/缩放地图，geo 地图和 map 地图不同步
- 问题 2:tooltip 内部写 html 代码事件支持、定位位置计算、背景图定位位置
- 问题 3:水波纹图标上面写文本
- 问题 4:注册中国地图时名字要叫`china`,其他名字会导致没有`南海诸岛`
- 问题 5: 多个弹框方案
  :::

## 1.单个地图-中国地图

- 改造 `geojson` 文件使中国地图外轮廓和中国地图放在一起形成一个图层实现地图 `3d` 效果
